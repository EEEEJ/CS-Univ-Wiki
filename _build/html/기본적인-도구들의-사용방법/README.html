기본적인 도구들의 사용방법¶

이 문서는 대학생이 되서 써야만 할 도구들에 대한 설명을 모아놓은 것 입니다. 기본적으로 여기에 열거된 도구들만 잘 써도 일상생활에서 하게 되는 컴퓨터를 이용한 반복작업이나 삽질을 엄청나게 줄일 수 있으며, 또 정말 많은 정보들을 얻고, 정리하고, 사용할 수 있게 될 것입니다.

인트로덕션에서 말 하였듯이,

  1. 컴퓨터 공학도라면 알았으면 좋겠는 것

  2. 대학생이라면 알았으면 좋겠는 것

이 두 가지에 대해서 이야기를 할 예정입니다.

즉, 대학생활에서 쓰게 될

  1. 읽을거리 관리 도구

    1. 메일

    2. RSS 리더

    3. 논문 관리 도구

  2. 개발 도구

    1. 터미널

    2. 에디터

    3. 패키지 관리자

    4. FTP

    5. ssh/telnet

    6. 버전 관리 도구

  3. 협업 도구

    1. Slack

    2. 일정 관리도구

    3. 문서 작성 도구

    4. 클라우드 동기화 서비스

    5. 쉬운 버전 관리 도구

    6. 이슈 트래커

  4. 삶에 도움이 되는 도구

    1. 패스워드 관리 도구

    2. 슬라이드 쉐어

  5. 애플 교도라면 써볼 만한 것들

    1. 정말

    2. 많음

에 대해서 이야기를 해보려고 합니다.
